# dotfiles
## about
personal packaging of important dot files
## usage
```
mkdir ~/repositories
cd ~/repositories/
git clone https://github.com/abutbul/dotfiles.git
cd dotfiles
./dotdep_vim.sh
./dotdep_tmux.sh
```
